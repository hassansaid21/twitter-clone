# Twitter Clone 🐦

A full-stack Twitter (X) clone built with **Next.js 15**, **Tailwind CSS**, and **TypeScript**.

## Features
- Modern UI inspired by X (Twitter)
- Responsive layout
- Real-time ready architecture
- Scalable project structure

## Tech Stack
- Next.js 15
- TypeScript
- Tailwind CSS

## Getting Started
```bash
npm install
npm run dev
